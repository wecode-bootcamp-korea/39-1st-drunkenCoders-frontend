# Drunken Coders

### 프로젝트 사이트 : 술담화

술담화는 전통주 구독 & 판매를 하고 있는 이커머스 사이트입니다.
최근 전통주에 대한 많은 관심과, 소규모 양조장 웹사이트들이 하나 둘 생겨나는 점을 볼 때 이번 프로젝트로 관련 경험을 쌓는 것도 좋을 것 같아 진행하였습니다.
